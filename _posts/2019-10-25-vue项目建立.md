vue项目建立

```shell
npm install vue-cli
```

或全局安装

```shell
npm install -g vue-cli
```


```shell
vue init webpack client
```

进行一些配置会生成client项目文件夹
只需修改src及index.html

生成打包文件夹dist

```shell
npm run build
```
生成的文件配置估计会有路径问题，需要简单修改。

ubuntu 16.04使用npm install时出错

提示找不到node命令

解决方法:
```shell
sudo apt install npm
```

n 模块是用来安装各种版本的 node 的一个工具。参数 -g 表示全局安装

```shell
sudo npm install n -g

sudo n lts

node -v
```
