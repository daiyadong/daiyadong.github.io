### 如果你没有vue-cli的话需要全局安装
```shell
cnpm install -g vue-cli
```

### 然后使用vue-cli来安装electron-vue的模板
```shell
vue init simulatedgreg/electron-vue my-project
```

### 安装依赖

```shell
cd my-project

yarn # or cnpm install
```

### 进入开发模式
```shell
yarn run dev # or cnpm run dev
```
